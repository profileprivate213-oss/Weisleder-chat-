Weisleder Chat — Multi-user (Local) Demo
=============================================
Frontend-only chat UI with **Sign up / Sign in**, **multiple users**, **1:1** and **group chats** — stored in your browser (localStorage).

How to use
1) Unzip the package.
2) Open `index.html` in your browser.
3) Click **Sign up** to create an account. Create several accounts to simulate multiple users on the same device.
4) Click **＋ New chat** to start a chat with one or more users.

What’s included
- Sign up / Sign in (local, no server)
- Multiple user accounts on one device
- Create 1:1 or group chats
- Messages persist locally
- Dark/Light theme
- Emoji panel & demo attachments
- Responsive layout

Going online (optional)
- Deploy folder to **GitHub Pages** or **Netlify** to share a link. This still remains single-device per user unless you add a backend.
- To sync across devices, connect to **Firebase** or **Supabase** later.
